# Proyecto final Pylatino 25
****

Este repositorio es mi entrega final para el programa `Pylatino 25` proporcionado por la Universidad de los Andes. AbyssBound es un juego de mazmorras en 2D, creado a partir de la librería `Pygame`.
Este proyecto permitió afianzar los conceptos vistos durante todo el curso, específicamente el manejo de librerías de gráficos, animaciones y manejo de archivos.
<br>

El arte del juego se realizó con el programa `LibreSprite`.

> [!NOTE]
> Música de fondo: "Trouble is Brewing" de David Fesliyan. [Enlace al sitio](https://www.fesliyanstudios.com/royalty-free-music/download/trouble-is-brewing/296)

> [!IMPORTANT]
> Al finalizar el juego y volver a iniciarlo, la dificultad y cantidad de enemigos aumentarán.

## Requisitos
- Python 3.10+ (Preferentemente Python 3.13)
- pip (incluido con Python)

## Instalación

```bash
# Clonar el repositorio
git clone <url>
cd <repo>
```
```bash
# Crear entorno virtual
python -m venv venv
```
```bash
# Activar entorno virtual
venv\Scripts\activate     # Windows

source venv/Scripts/activate  # Linux/Mac
```
```bash
# Instalar dependencias
pip install -r requirements.txt
```
```bash
# Ejecutar el juego
python -m src.game.Main
```
